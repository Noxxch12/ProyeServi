AD

1. Descarga de la ISO de Windows Server 2022

Lo primero que debemos hacer es descargar la imagen ISO de Windows Server 2022. Se recomienda elegir la versión en tu idioma para facilitar la configuración.
Descaga aquí

2. Montar la ISO en una Máquina Virtual

Una vez descargada la ISO, procedemos a montarla en una máquina virtual (puede ser en plataformas como VirtualBox, VMware o Hyper-V).

3. Instalación de Windows Server

Con la ISO montada, iniciamos la instalación del sistema operativo. Al finalizar, accedemos al Administrador del Servidor y damos clic en la opción "Agregar roles y características".

4. Asistente de configuración

Se abrirá un asistente de instalación. Avanzamos haciendo clic en "Siguiente" en cada paso, dejando la configuración por defecto. Al llegar a la ventana de selección de roles, activamos la casilla "Servicios de dominio de Active Directory (AD DS)".

5. Confirmación e instalación del rol

Confirmamos la instalación y esperamos a que el proceso finalice.

6. Creación de dominio

Una vez instalado el rol AD DS, lo siguiente es crear un dominio y configurar el Active Directory. Para esta parte, puedes seguir el siguiente tutorial en video, que fue el que utilicé como guía:

Una vez que tenemos el dominio y Active Directory configurados correctamente, el siguiente paso es crear una política de grupo para bloquear el acceso a Microsoft Edge. Para esto, seguí el siguiente video tutorial:

8. Aplicación de la política

Ya con la GPO creada y aplicada correctamente, al probar la apertura de Microsoft Edge, el sistema impide el acceso, lo que confirma que la política fue aplicada con éxito.
