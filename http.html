1. Instalación del sistema operativo

Primero, debemos instalar Ubuntu Server en nuestra máquina virtual. Asegúrate de seguir todos los pasos necesarios para completar la instalación correctamente..

2. Actualización del sistema

Una vez instalado Ubuntu, actualizamos la lista de paquetes y el sistema con el siguiente comando:
sudo apt update && sudo apt upgrade -y

3. Instalación de Apache

Para instalar el servidor Apache, ejecuta el siguiente comando:
sudo apt install apache2 -y
Este comando descargará e instalará Apache automáticamente.

4. Verificar el estado del servicio

Después de instalar Apache, verifica si el servicio está activo con:
sudo systemctl status apache2
Si el estado no aparece como "active (running)", puedes iniciar Apache manualmente con:
sudo systemctl start apache2

5. Comprobar funcionamiento del servidor

Abre un navegador web y escribe la dirección IP de tu servidor Ubuntu. Si Apache está funcionando correctamente, verás una página de bienvenida
