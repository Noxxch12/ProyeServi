FTP

1. Instalar Ubuntu Server

Primero, debemos instalar Ubuntu Server en una máquina virtual.

2. Actualizar los repositorios

Una vez instalado, actualizamos los repositorios con el siguiente comando:
sudo apt update && sudo apt upgrade -y

3. Instalar el servidor FTP

Instalamos el servidor FTP usando el siguiente comando:
sudo apt install vsftpd

4. Configurar el servidor FTP

Editamos el archivo de configuración con el siguiente comando:
sudo nano /etc/vsftpd.conf
Buscamos la línea write_enable=YES y nos aseguramos de que no esté comentada (eliminamos el símbolo # si lo tiene).

5. Reiniciar el servicio

Aplicamos los cambios reiniciando el servicio con:
sudo systemctl restart vsftpd.service

6. Crear un nuevo usuario

Creamos un usuario con el comando:
adduser nombre_de_usuario Completamos los campos solicitados.

7. Verificar que el usuario fue creado

Usamos el siguiente comando para verificar:
sudo cat /etc/passwd

8. Habilitar el servicio al iniciar

Para que el servidor FTP se inicie automáticamente al encender la máquina, ejecutamos el comando:
sudo systemctl enable vsftpd.service

9. Instalar FileZilla

Descargamos e instalamos FileZilla en nuestra computadora.
Descarga aquí

10. Conectar a través de FileZilla

Abrimos FileZilla y colocamos los siguientes datos:
Host: Dirección IP del servidor
Nombre de usuario: Usuario creado
Contraseña: Contraseña del usuario
Puerto: 21

11. Crear un nuevo directorio y crear un archivo

Dentro de la carpeta del usuario FTP, creamos un nuevo directorio.
Creamos un archivo nuevo dentro del directorio del usuario FTP (por ejemplo, en /home/ftpuser).

13. Verificar el funcionamiento

Por último, accedemos a la carpeta del usuario para verificar que el archivo creado esté visible, lo que confirma que el servidor FTP está funcionando correctamente.
