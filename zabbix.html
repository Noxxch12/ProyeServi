ZABBIX

1. Descarga de Ubuntu Server

Primero, descarga la imagen ISO de Ubuntu Server desde el sitio oficial:
Descarga aquí

2. Instalación en máquina virtual

Una vez descargado Ubuntu Server, instálalo en una máquina virtual (puedes usar VirtualBox, VMware, etc.). Durante la instalación, asegúrate de tener acceso a la red.

3. Configuración inicial del servidor

Después de instalar Ubuntu Server, accede a la terminal e introduce los siguientes comandos para preparar el entorno:
sudo apt update-Actualiza la lista de paquetes
sudo apt upgrade-Actualiza los paquetes instalados
sudo apt install net-tools-Instala herramientas de red como ifconfig
ifconfig-Muestra la IP del servidor
sudo apt install openssh-server-Instala el servidor SSH
sudo systemctl status ssh-Verifica que el servicio SSH esté activo

4. Conexión SSH desde tu máquina física

Con la IP que obtuviste con ifconfig, crea una conexión SSH desde tu máquina principal hacia el servidor, por ejemplo:
ssh noezavala@192.168.1.20.
Esto te permite administrar el servidor desde una terminal externa, facilitando la instalación de herramientas como Zabbix.

5. Instalación de Zabbix Server

Ahora, accede al sitio oficial para descargar la versión adecuada de Zabbix:
Descarga aquí
Selecciona.
Sistema operativo:Ubuntu 24.04
Componentes: Server, Frontend, Agent
Base de datos: MySQL
Servidor web: Apache
Luego, sigue los pasos que te proporciona el sitio para instalarlo.
Recomendaciones importantes:.
Ten paciencia durante la instalación.
No interrumpas ni cierres la terminal.
Copia y pega los comandos desde tu terminal SSH para mayor comodidad y rapidez.

6. Configuración del agente Zabbix en otra máquina

Necesitas una segunda máquina (puede ser virtual o física) que será monitoreada por Zabbix. En este ejemplo, se utilizó Windows 10.
Descarga el agente desde el sitio oficial:
Descarga aquí
Instala el agente en Windows y en la ventana de configuración introduce la IP del servidor Zabbix cuando lo solicite.

7. Agregar un nuevo host en Zabbix

Desde la interfaz web de Zabbix (accediendo con http://ip-del-servidor/zabbix): Inicia sesión como administrador.
Haz clic en la pestaña "Monitorización" > "Equipos"
En la esquina superior derecha, haz clic en “Crear”.

8. Configurar el nuevo host

En la ventana de creación:
Asigna un nombre descriptivo al host.
Introduce la IP de la máquina monitoreada (la segunda máquina).
Selecciona el grupo adecuado (puede ser “Windows hosts”).
Asegúrate de usar el agente Zabbix como método de monitoreo.
Guarda los cambios haciendo clic en “Add”.

9. Verificar monitoreo

Una vez agregado el host, en el panel de Zabbix deberías ver la actividad y los datos de monitoreo de la máquina Windows. Si todo está configurado correctamente, aparecerán los primeros valores y gráficos.
