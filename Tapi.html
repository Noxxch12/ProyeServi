TAPI

1. Descarga de FreePBX

Accede al sitio oficial de FreePBX para descargar la imagen ISO:
Descaga aquí

2. Montar la ISO en una Máquina Virtual

Crea una nueva máquina virtual y monta la ISO de FreePBX

3. Inicio de la Instalación

Al iniciar la máquina virtual, se te pedirá crear una contraseña para el usuario root. Escríbela y continúa.

4. Configuración Inicial

Completa las configuraciones iniciales que el sistema te solicite y selecciona “Done” al finalizar.

5. Instalación del Sistema

Espera a que FreePBX se instale completamente. Este proceso puede tardar varios minutos.

6. Reinicio del Sistema

Una vez finalizada la instalación, reinicia la máquina virtual. Cuando arranque de nuevo, apágala.

7. Configuraciones Post-Instalación

Antes de encender la máquina otra vez, realiza lo siguiente:
Quita la ISO del sistema desde la configuración de almacenamiento de la VM.
Cambia la configuración de red a:.
Tipo: Adaptador puente
Modo promiscuo: Permitir todas las máquinas virtuales

8. Inicio y Acceso

Enciende la máquina virtual nuevamente.
Inicia sesión como root usando la contraseña que configuraste.
Copia la dirección IP mostrada en la terminal.

9. Acceso desde el Navegador

Abre tu navegador web e introduce la IP copiada.
Llena el formulario de instalación de FreePBX y haz clic en “Setup System”.

10. Activación y Configuración de FreePBX

Accede a la opción “FreePBX Administration”.
Selecciona el idioma preferido y haz clic en “Submit”.
En las siguientes ventanas, acepta todas las configuraciones con “Yes”, “Continue” o “Next”.

11. Creación de Extensiones

Dirígete al menú Applications > Extensions.
Selecciona el tipo de extensión: SIP [chan_pjsip].
Completa los siguientes campos:.
User Extension
Display Name
Secret (contraseña)
Haz clic en Submit para guardar.
Repite este procedimiento para crear un total de tres extensiones.
Al finalizar, haz clic en “Apply Config” para aplicar los cambios.

12. Descarga e Instalación de Softphones

Descarga e instala las siguientes aplicaciones en tu computadora:
Descarga aquí zoiper
Descarga aquí 3cx

13. Configurar 3CX

Abre la aplicación de 3CX. Crea una cuenta usando los datos de la primera extensión (user, contraseña y IP del servidor).

14. Configurar Zoiper5

Al abrir Zoiper5:
Ingresa la extensión y contraseña del usuario 2.
Introduce la IP del servidor FreePBX.

15. Prueba de Llamada

Abre el Dial Pad en cualquiera de los softphones.
Marca la extensión del otro usuario.
Si todo está configurado correctamente, ¡ambos usuarios podrán comunicarse sin problemas!
